# academic-writing-clean

Use when coordinating English-language academic writing and revision for medical or biomedical manuscripts. Acts as the router for abstract, background, methods, results, and discussion skills while enforcing evidence boundaries, source-of-truth rules, and anti-AI style checks.

- Version: `2.0.0`
- Source of truth: `SKILL.md`
